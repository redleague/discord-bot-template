# About

An open-source discord-bot template music

# Features 

> ✅ Easy-to-use <br>
> ✅ Stable

# Installation
Recommended version Node.js 14
Node.js 12, is also supported.

To install all the packages just use:

```bash
npm install
```
# Run

```bash
node app.js
```
